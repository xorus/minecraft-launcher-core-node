description

description: string | object
  • Defined in packages/resourcepack/format.ts:103

Text that will be shown below the pack name in the resource pack menu. The text will be shown on two lines. If the text is too long it will be cut off.

Contains a raw JSON text object that will be shown instead as the pack description in the resource pack menu. Same behavior as the string version of the description tag, but they cannot exist together.[

pack_format

pack_format: number
  • Defined in packages/resourcepack/format.ts:95

Pack version. If this number does not match the current required number, the resource pack will display an error and required additional confirmation to load the pack. Requires 1 for 1.6.1–1.8.9, 2 for 1.9–1.10.2, 3 for 1.11–1.12.2, and 4 for 1.13–1.14.4.
